# 感謝你花點時間了解我

我主修電子工程, 小時候在雲林鄉下長大, 從台中高工電子科爬到台北科大電子系, 一路上跟著許多人學到了各種知識, 大學期間大二進入多媒體系統實驗室, 正式開始了程式生涯 ,隨著參與開發科技部專案 italkutalk 計畫, 接觸到了 API 串接與 WebRTC 的應用, 那時主要負責轉換 Andriod 系統至 iOS 平台上, 所以在 java 與 swift 上有了點研究, 也要習慣不同開發方式的差異, 還有版本的支援程度

<br />

後來大學專題使用 Line bot 做為一個快速擴展產品曝光的原型, 最初構想為之前接觸 swift 與 java, 知道在兩大平台開發的時間及難易度, 因此以 node.js 及 express 快速建立後端使用者服務及自然語言庫 API.AI(現為 Dialogflow)橋接, 使機器人回話能有一定人性化

<br />
之後被教授採用做為科技部專案 BlueNet 計畫的延伸, 在原有 ios 與 Andriod 端以外, 增加 line 官方帳號, 當時負責開發手機版網頁叫車介面與 chatbot 主程式, 提供 Line 上使用者的註冊流程, 透過聊天室對話及簡訊驗證碼認證, 綁定舊用戶或創建新用戶, 綁定後進入官方帳號選單的功能性網頁時, 前端網頁就以 Line userid 對後台請求通行證, 這樣使用者就可以用 LINE 的身分進行叫計程車了

# 進入職場的心情

畢業後服完兵役也該準備人生下一個階段了, 想想也是從 20 歲開始正式自學程式, 當初也是一股腦地投入, 突然就面對 Objective-C 的震撼, 還好當初沒有放棄, 後來因為實驗室缺人手而學習到後端的部分, 才發現從後端學習的話, 相對較慢才能看到成果, 很大原因都是因為前端及 UI, 等到了前端後才發現 UX 的重要性, 直接影響到訂閱人數及評論, 看來前端想精通, 前端工程及使用者體驗都要做好, 未來希望成為一個掌握 UI / UX 的前端開發者, 之後再慢慢朝向全端開發邁進

我希望記錄一下開始進入職場的心情, 以後可以好好回顧
