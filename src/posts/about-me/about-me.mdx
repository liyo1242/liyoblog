# 感謝你花點時間了解我

我主修電子工程, 小時候在雲林鄉下長大, 從台中高工電子科爬到台北科大電子系, 一路上跟著許多人學到了各種知識, 披荊斬棘活到了現在, 大學期間大二進入多媒體實驗室, 參與開發科技部專案 italkutalk 計畫, 剛開始負責 andriod 程式碼轉換 ios, 當時使用語言為 Swift 做基礎

<br />
後來大學專題使用 Line bot 做為一個快速擴展產品曝光的原型, 最初構想為之前接觸 Objective-C 與 java, 知道在兩大平台開發的時間及難易度, 因此以 node.js 及 express 快速建立後端使用者認證及 API.AI(現為 Dialogflow)橋接, 使機器人回話能有一定人性化,
<br />
之後被教授採用做為科技部專案 BlueNet 計畫的延伸, 在原有 ios 與 Andriod 端以外, 增加 line 官方帳號, 當時負責開發手機版網頁叫車介面, 與 google map API 做結合, 後來負責手機版小秘書功能, 結合 google calendar API 讓使用者的叫車紀錄跟 google calendar 及 line 的提醒有更好的效果
<br />
